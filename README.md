# Prelucrarea Imaginilor

Acest proiect este un editor de imagini Bitmap (.bmp) care permite aplicarea filtrelor de culoare și analiza geometrică a obiectelor.

📋 Detalii Tehnice:
Fișier principal: App.py (conține întreaga logică de UI și procesare).
Versiune Python recomandată: Python 3.13.12.

💡 Funcționalități incluse:
Filtre Color: RGB, CMYK, HSV, YUV, YCbCr, Binarizare și Negativ.
Analiză Obiect: Calcul centru de greutate , momente centrale, matrice de covarianță și proiecții pe axe.
Utilitare: Vizualizare histogramă și export fișier editat.
